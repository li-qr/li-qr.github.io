# 设计模式

## 创造型模式

+ abstract factory（抽象工厂）
+ builder（生成器）
+ factory method（工厂方法）
+ prototype（原型）
+ singleton（单例）

## 结构型模式

+ adapter（适配器）
+ bridge（桥接）
+ composite（组成）
+ decorator（装饰）
+ facade（外观）
+ flyweight（享元）
+ proxy（代理）

## 行为模式

+ chain of responsibility（责任链）
+ command（命令）
+ interpreter（解释器）
+ iterator（迭代器）
+ mediator（中介者）
+ memento（备忘录）
+ observer（观察者）
+ state（状态）
+ strategy（策略）
+ template method（模板方法）
+ visitor（访问者）
